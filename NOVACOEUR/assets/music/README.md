# Musique - NOVACOEUR

Ce dossier contient les fichiers musicaux pour les pages d'amour.

## Structure

Les utilisateurs peuvent charger leur propre musique directement depuis l'interface admin.
Les fichiers MP3, WAV, et OGG sont acceptés.

## Format acceptés
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- WebM (.webm)

**Note:** Les musiques présélectionnées ont été supprimées. Veuillez charger votre propre musique depuis l'interface admin.
